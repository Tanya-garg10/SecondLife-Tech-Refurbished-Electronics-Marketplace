# Project Description:
SecondLife Tech is a full-stack marketplace for buying and selling refurbished electronics like laptops, phones, and gadgets. It includes AI-powered quality verification, price comparison, delivery tracking, and review analysis to ensure a safe and seamless experience for both buyers and sellers.

- Resale Price Predictor
